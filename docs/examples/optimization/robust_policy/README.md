Robust policy optimization
==========================

Find a single policy that is feasible across multiple SSP2 climate futures (SSP2-1.9,
SSP2-2.6, SSP2-3.4) simultaneously. Two variants: trend resource allocation and
low-demand formulation.

Each run script executes a single robust variant. The comparison script gathers both
and plots multi-scenario results.
