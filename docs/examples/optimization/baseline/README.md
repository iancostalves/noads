Baseline (fossil kerosene only)
===============================

Optimization using only fossil kerosene — no alternative fuels, no new aircraft types.
This establishes the counterfactual: what happens if aviation does nothing?

Each run script optimizes for a single background scenario (SSP1-1.9, SSP2-2.6, or
SSP5-4.5) across three technology levels (lower, mid, upper). The comparison script
gathers all results and plots cross-scenario trends.
