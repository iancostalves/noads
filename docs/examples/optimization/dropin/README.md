Drop-in SAF scenarios
=====================

Optimization with drop-in sustainable aviation fuels (HEFA, Fischer–Tropsch, ATJ,
Power-to-Liquid), all kerosene-compatible. Four variants are tested under SSP2-2.6:
baseline reference, trend resource allocation, preferential energy allocation, and
low-demand formulation.

Each run script executes a single variant across three technology levels. The comparison
script gathers all variants and plots trend and jet fuel composition comparisons.
