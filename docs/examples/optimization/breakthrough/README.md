Breakthrough aircraft scenarios
===============================

Optimization with all propulsion architectures unlocked: hydrogen gas turbine, hydrogen
fuel cell, battery-electric, plus drop-in SAF. Four variants are tested under SSP2-2.6:
baseline reference, trend, preferential energy, and low-demand formulation.

Each run script executes a single variant across three technology levels. The comparison
script gathers all variants and plots trends, jet fuel composition, and fleet energy
carrier breakdowns.
