Optimization
============

The scenario families studied in the paper, each optimized with the same
gradient-based formulation: the fossil-only baseline, drop-in sustainable
aviation fuels, breakthrough aircraft architectures, and a policy robust to
several climate futures, followed by a validation against the AR6 scenario
ensemble.

The ``run_*.py`` scripts reproduce each optimization from scratch (hours of
compute); the ``plot_compare_*.py`` scripts load the pre-computed optima shipped
with the repository and reproduce the comparison figures.
